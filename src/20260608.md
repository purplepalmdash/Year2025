### 1. Build OpenEuler Kernel
Find the committment under `https://gitee.com/src-openeuler/kernel/commit/59664facbc4bba10fe81fc251a9f2f7c4406a31e `, then:      

```
git clone https://atomgit.com/src-openeuler/kernel.git
cd kernel
# 切换到分支
git checkout openEuler-22.03-LTS-SP1
# 强制重置分支到指定 commit（会丢弃该 commit 之后的所有本地修改/提交）
git reset --hard 59664facbc4bba10fe81fc251a9f2f7c4406a31e
```
Install dependencies:     

```
sudo dnf install rpm-build rpmdevtools dnf-plugins-core
sudo dnf builddep kernel.spec 
```

As mock setup the rpmdev tree:     

```
su - mock
cd /home/mock
rpmdev-setuptree

cd ~/Code/kernel
cp kernel.spec ~/rpmbuild/SPECS/
cp ~/Code/kernel/* ~/rpmbuild/SOURCES
```
Fetch the kernel:      

```
git clone https://gitee.com/openeuler/kernel.git
cd kernel
 git tag | grep 5.10.0-136.92
git checkout 5.10.0-136.92.0
cd ..
tar czvf kernel.tar.gz kernel
```
Move the kernel.tar.gz to `~/rpmbuild/SOURCES`.      

Then build:       

```
cd ~/rpmbuild/SPECS
time rpmbuild -ba kernel.spec
```
