### 2. Auto-load binder
Via:      

```
test@testflex2:~$ cat /etc/modprobe.d/binder_linux.conf
# 为 binder_linux 模块设置参数
options binder_linux devices="binder,hwbinder,vndbinder"
test@testflex2:~$ cat /etc/modules-load.d/binder_linux.conf
binder_linux
```
