# 20260520
### 1. Install pi:      
Install with one line:    

```
curl -fsSL https://pi.dev/install.sh | sh
pi 
```
Install nim plugin:       

```
 pi -e https://github.com/xRyul/pi-nvidia-nim
vim ~/.bashrc
# 推荐（pi-nvidia-nim 扩展优先使用这个）
export NVIDIA_NIM_API_KEY=nvapi-你的密钥在这里

# 或者使用这个（也兼容）
export NVIDIA_API_KEY=nvapi-你的密钥在这里
```
