### 2. waydroid-ATV(lineageOS23.2)
Steps:      

```
cd waydroid_lineageos23.2/
 export REPO_URL='https://mirrors.bfsu.edu.cn/git/git-repo'
 repo init -u https://github.com/LineageOS/android.git -b lineage-23.2 --git-lfs
 repo sync build/make
 export REPO_URL='https://mirrors.bfsu.edu.cn/git/git-repo'
 wget -O - https://raw.githubusercontent.com/WayDroid-ATV/android_vendor_waydroid/lineage-23.2/manifest_scripts/generate-manifest.sh | bash
 cp .repo/manifests/default.xml  ~/default.xml.back
 vim .repo/manifests/default.xml 
############## Changs to local repository
repo sync 
############## Will fail, cannot repo sync prebuilts/misc(3918627)
grep -rn "3918627" .repo/
vim .repo/manifests/default.xml +853
cp ~/default.xml.back  .repo/manifests/default.xml 
repo sync --force-sync prebuilts/misc
repo sync
```
Build with docker:     

```
source  build/envsetup.sh 
apply-waydroid-patches 
git config --global --add safe.directory  '*'
cat build/make/core/build_id.mk
lunch lineage_waydroid_tv_x86_64-bp4a-userdebug
### Make sure you enable the redroid related patch.
time make systemimage -j$(nproc --all) && time make vendorimage -j$(nproc --all)
get_build_var TARGET_COPY_OUT_VENDOR
The result is vendor
```
Extract the docker images via:     

```
sudo mount system.img system -o ro && sudo mount vendor.img vendor -o ro && sudo tar --xattrs -c vendor -C system --exclude="./vendor" . | sudo docker import --platform=linux/amd64 -c 'ENTRYPOINT ["/init", "ro.hardware.hwcomposer=redroid", "ro.adb.secure=0", "ro.hardware.hwcomposer=waydroid"]' - ownwaydroid232 && sudo umount system && sudo umount vendo
```
