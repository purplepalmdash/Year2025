# 20260526
### 1. n100 wol
Configure via:      

```
test@n100:~$ sudo ethtool enp2s0 | grep 'Wake-on'
	Supports Wake-on: pumbg
	Wake-on: d
test@n100:~$ cat <<EOF | sudo tee /etc/systemd/system/wol.service
[Unit]
Description=Enable Wake On LAN
After=network.target

[Service]
Type=oneshot
ExecStart=/sbin/ethtool -s enp2s0 wol g
RemainAfterExit=yes

[Install]
WantedBy=multi-user.target
EOF
test@n100:~$ sudo vim /etc/systemd/system/wol.service 
test@n100:~$ sudo systemctl daemon-reload
test@n100:~$ sudo systemctl enable --now wol.service
Created symlink /etc/systemd/system/multi-user.target.wants/wol.service → /etc/systemd/system/wol.service.
test@n100:~$ sudo ethtool enp2s0 | grep 'Wake-on'
	Supports Wake-on: pumbg
	Wake-on: g

```
Wakeup via:     

```
ssh root@192.168.1.172 'wakeonlan -i  10.171.172.255 xx:xx:xx'
```
