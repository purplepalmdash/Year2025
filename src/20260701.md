### 1. ast/amdgpu issue
Under x11, though disabled ast(blacklisted ast), the Xorg still use ast card for displaying.     

Using following configuration for letting amdgpu for the primary gpu:     

```
sudo nano /etc/X11/xorg.conf.d/10-amdgpu.conf

Section "Device"
    Identifier "AMD Graphics"
    Driver "amdgpu"
    BusID "PCI:8:0:0"
    Option "PrimaryGPU" "yes"
EndSection

Section "Device"
    Identifier "ASPEED Graphics"
    Driver "ast"
    Option "PrimaryGPU" "no"
EndSection
```
