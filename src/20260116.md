# 20260116
### 1. QT开发环境

```
  第一步：安装 Visual Studio 2022 (获取 C++ 编译器)

  Visual Studio 为我们提供核心的 C++ 编译器（MSVC）、调试器和构建工具。

   1. 下载 Visual Studio Community 2022:
       * 访问官网下载页面：https://visualstudio.microsoft.com/vs/community/ (https://visualstudio.microsoft.com/vs/community/)
       * 点击 "免费下载"，会下载一个名为 VisualStudioSetup.exe 或类似的文件。

   2. 运行安装程序:
       * 双击运行下载的安装程序。它会先下载和准备安装所需的文件，请稍等片刻。

   3. 选择工作负载 (最关键的一步):
       * 准备完成后，您会看到一个标题为“工作负载”的界面，这里有很多开发选项。
       * 您必须找到并勾选 “使用 C++ 的桌面开发” 这一项。
       * !Visual Studio 工作负载选择 (https://learn.microsoft.com/zh-cn/cpp/get-started/media/vs-installer-workloads.png?view=msvc-170)
       * 在右侧的“安装详细信息”中，您可以保持默认勾选的选项不变。
       * 点击右下角的 “安装” 按钮。

   4. 完成安装:
       * 安装过程会持续一段时间。安装完成后，您可以关闭 Visual Studio。我们暂时不需要打开它，只需要它的编译器。

  ---

  第二步：安装 Qt (获取 UI 库)

  Qt 为我们提供构建图形用户界面所需的所有库文件（DLLs）。

   1. 注册 Qt 账户:
       * Qt 的开源版本需要一个免费的 Qt 账户才能下载。请先访问这里注册：https://login.qt.io/register (https://login.qt.io/register)

   2. 下载 Qt Online Installer:
       * 访问官网下载页面：https://www.qt.io/download-open-source (https://www.qt.io/download-open-source)
       * 点击 "Download the Qt Online Installer"，下载安装程序。

   3. 运行安装程序并登录:
       * 运行下载的安装程序，并使用您刚刚注册的 Qt 账户登录。

   4. 选择组件 (最关键的一步):
       * 在安装过程中，您会到达一个“选择组件 (Select Components)”的界面。
       * 请勾选 "Custom installation" (自定义安装)。
       * 找到最新的 Qt 版本，例如 "Qt 6.5.3" (版本号可能不同)，点开它前面的小三角。
       * 在展开的列表中，找到并只勾选与您 Visual Studio 版本匹配的那一项，它的名字应该是 "MSVC 2019 64-bit" 或 "MSVC 2022 64-bit"。
           * 为了节省空间，请确保只勾选这一个！ 其他如 MinGW、Android、Sources 等都可以不勾选。
       * 展开下方的 "Tools" (工具) 列表，确保 "CMake" 是勾选的（通常默认会勾选）。
       * 点击 "Next" (下一步)，同意许可协议，然后点击 "Install" (安装)。

  ---

  第三步：安装 Vcpkg 并获取依赖库

  Vcpkg 是一个命令行工具，它能帮我们自动下载和编译 spice-gtk 和 libusb 这两个项目依赖库。

   1. 准备文件夹并下载 vcpkg:
       * 在您的电脑上创建一个您喜欢的位置，例如 C:\dev。
       * 打开 PowerShell (您可以在开始菜单中搜索 PowerShell)。
       * 在 PowerShell 中，进入您创建的目录，并运行以下命令：
   1         cd C:\dev
   2         git clone https://github.com/microsoft/vcpkg.git
   3         .\vcpkg\bootstrap-vcpkg.bat

   2. 安装依赖库:
       * 上一步完成后，继续在同一个 PowerShell 窗口中运行以下命令。这个过程会花费很长时间（可能半小时到数小时），因为它需要从头编译很多库，请保持网络连接并耐心等待。

   1         .\vcpkg\vcpkg.exe install spice-gtk:x64-windows libusb:x64-windows

  ---

  当以上所有三个步骤都成功完成后，您就拥有了一个完整、稳定、可以编译我们这个项目的专业开发环境了。

  请在完成后通知我，我将指导您完成最后的编译步骤。

```
