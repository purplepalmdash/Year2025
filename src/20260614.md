### 1. ccache
Set ccache via:    

```
cd /mnt/lineage/
source build/envsetup.sh 
git config --global --add safe.directory "*"
source build/envsetup.sh 
apply-waydroid-patches 
export USE_CCACHE=1
export CCACHE_COMPRESS=1
export CCACHE_DIR="/ccache"
export CCACHE_EXEC=/usr/bin/ccache
${CCACHE_EXEC} -M  50G
```
