### 2. pi dev console context
via:      

```
npm install -g context-mode
pi install npm:context-model npm:context-mode
cat ~/.pi/agent/settings.json
Add:    

  "mcpServers": {
    "context-mode": {
      "command": "context-mode"
    }
  }

```
