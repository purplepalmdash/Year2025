D [05/Nov/2025:09:05:20 +0800] [Job 6] FP-630KII: error while loading shared libraries: libcupsfilters.so.1: cannot open shared object file: No such file or directory
这行日志完美地解释了为什么 hotpotfilter 会失败。
    hotpotfilter 启动时，它尝试加载它所依赖的共享库 libcupsfilters.so.1。
    系统报告该文件不存在。
    因此，hotpotfilter 进程立即崩溃，导致了 status 127 和整个打印作业的失败。
