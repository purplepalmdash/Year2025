### 3. Fedora38 xorg Configuration
cmdline should be:     

```
$ cat /proc/cmdline
BOOT_IMAGE=(hd1,gpt2)/vmlinuz-6.2.9-300.fc38.x86_64 root=UUID=2ae22a56-91c8-4ac3-8ded-aa913ce66ced ro rootflags=subvol=root rhgb quiet initcall_blacklist=simplefb_init initcall_blacklist=ast sysfb.simplefb=0 initcall_blacklist=simpledrm_platform_driver_init modprobe.blacklist=ast
```
Then add a xorg conf file:     

```
[test@fedora ~]$ cat /etc/X11/xorg.conf.d/10-amdgpu.conf 
Section "ServerLayout"
    Identifier     "Default Layout"
    Screen      0  "AMD-Screen" 0 0
EndSection

Section "Device"
    Identifier  "AMD-GPU"
    Driver      "amdgpu"
    # 锁定你的 AMD 显卡总线位置
    BusID       "PCI:8:0:0"
EndSection

Section "Monitor"
    Identifier  "Monitor0"
EndSection

Section "Screen"
    Identifier  "AMD-Screen"
    Device      "AMD-GPU"
    Monitor     "Monitor0"
    DefaultDepth 24
    SubSection "Display"
        Viewport   0 0
        Depth     24
    EndSubSection
EndSection
```
