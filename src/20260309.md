# 20260309
### 1. Google-chrome issue on ubuntu24.04
After upgrading to ubuntu24.04, google-chrome fails to work:     

Issue is listed as:      

```
$ google-chrome-stable 
[478277:478277:0309/083253.865703:FATAL:sandbox/linux/services/credentials.cc:131] Check failed: . : 权限不够 (13)
追踪或断点陷阱 (核心已转储)
```
Solved via:    

```
$ sudo sysctl -w kernel.apparmor_restrict_unprivileged_userns=0

echo 'kernel.apparmor_restrict_unprivileged_userns = 0' | sudo tee /etc/sysctl.d/60-apparmor-namespace.conf
```
