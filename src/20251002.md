### 1. ubuntu1804 build libvirt

```
apt install -y xsltproc python3-docutils libtirpc-dev libgnutls28-dev libxml2-dev libpciaccess-dev libyajl-dev python3-pip
cd libvirt-8.0.0
sudo python3 -m pip install meson -i https://pypi.tuna.tsinghua.edu.cn/simple
/usr/local/bin/meson build -Dsystem=true -Ddriver_qemu=enabled -Ddriver_interface=enabled -Ddriver_libvirtd=enabled -Ddriver_remote=enabled -Ddriver_network=enabled --prefix=/usr

```
