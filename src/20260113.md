### 1. docker run rerank model
run docker instance:    

```
sudo docker run --name xinference -d -p 9997:9997   --gpus all   -e XINFERENCE_MODEL_SRC=modelscope   -v $PWD/model_data:/root/.xinference/cache   xprobe/xinference:latest xinference-local -H 0.0.0.0 --port 9997
```
Download `bge-reranker-large`:     

```
curl -X POST "http://127.0.0.1:9997/v1/models" -H "Content-Type: application/json" -d '{
           "model_engine": "transformers",
           "model_name": "bge-reranker-large",
           "model_type": "rerank"
         }'
```
Get the result:     

```
curl http://localhost:9997/v1/models
{"object":"list","data":[{"id":"bge-reranker-large","object":"model","created":0,"owned_by":"xinference","model_type":"rerank","address":"0.0.0.0:35277","accelerators":["0"],"type":"normal","model_name":"bge-reranker-large","language":["en","zh"],"model_revision":"v0.0.1","replica":1}]}

```
