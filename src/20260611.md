# 20260611
### 1. cpulimit
Install:     

```
sudo apt install -y cpulimit
```
Limit via:     

```
sudo crontab -e

@reboot while ! pgrep -x "kkk-mon" > /dev/null; do sleep 5; done; /usr/bin/cpulimit -e kkk-mon -l 5 -b
@reboot while ! pgrep -x "fuckddh" > /dev/null; do sleep 5; done; /usr/bin/cpulimit -e fuckddh -l 5 -b

```
