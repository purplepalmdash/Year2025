### 1. win10 autologin
Steps:     

```
选择“开始”，然后选择“运行”。
在 “打开 ”框中，键入 Regedit.exe，然后按 Enter。
在注册表中找到HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon子项。
在“编辑”菜单上，选择“新建”，然后指向“字符串值”。
键入 AutoAdminLogon，然后按 Enter。
双击“AutoAdminLogon”。
在 “编辑字符串 ”对话框中，键入 1 ，然后选择“ 确定”。
双击 DefaultUserName 条目，键入用户名，然后选择“ 确定”。
双击 DefaultPassword 条目，键入密码，然后选择“ 确定”。
如果 DefaultPassword 值不存在，则必须添加该值。 要添加该值，请执行以下步骤：
在“编辑”菜单上，选择“新建”，然后指向“字符串值”。
键入 DefaultPassword，然后按 Enter。
双击“DefaultPassword”。
在 “编辑字符串 ”对话框中，键入密码，然后选择“ 确定”。
```
### 2. qcow2 convert
Convert to vmdk format:      

```
qemu-img convert -f qcow2 -O vmdk \
         -o compat6,subformat=streamOptimized \
         ubuntu22.04.qcow2 ubuntu22.04-vmware.vmdk
```
Convert to virtualbox format:     

```
qemu-img convert -f qcow2 -O vdi your-image.qcow2 your-image-virtualbox.vdi
```
