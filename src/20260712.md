### 1. nomachine configuration
archlinux(7.x kernel), solved via:      

```
~/.nx/config/player.cfg

（如果没有这个文件，就新建一个）添加或修改：

Enable hardware accelerated decoding 0

$ cat ~/.nx/config/player.cfg| grep 'hardware'
  <option key="Enable hardware accelerated decoding" value="false" />
```
Then restart the nomachine, this time you could use nx for connecting remote machine.
