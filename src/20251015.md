### 2. raw printer config
Difference between raw/lpd:    

```
协议	端口	连接地址/URI 格式	优点
RAW	9100	socket://<IP地址>:9100	非常简单，通用，高效，是目前的主流选择。
LPD	515	lpd://<IP地址>/<队列名>	兼容性好，尤其是在一些旧设备或混合环境中。
```
