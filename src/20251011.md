### 2. ipp-usb build(ubuntu20.04)
Build steps:    

```
git clone https://github.com/OpenPrinting/ipp-usb.git
go get -u github.com/jstemmer/gotags
export PATH=$PATH:$(go env GOPATH)/bin
sudo cp /home/test/go/bin/gotags /usr/bin
cd ipp-usb
make
sudo make install
sudo reboot
```
