### 1. redroid普通用户拉起方法
root用户下，将普通用户加到docker组内：     

```
usermod -aG docker test
```
普通用户`test`重新登陆(re-login)后, 使用`id`检查是否已正确添加docker组，之后即可正常启动redroid容器：      

```
[test@redroid ~]$ id
uid=1000(test) gid=1000(test) groups=1000(test),10(wheel),986(docker)
[test@redroid ~]$ docker run -itd -v ~/datano:/data -p 5333:5555 --storage-opt size=15G --name redroidastest --memory="8192m" --cpus="4" --privileged  redroid/redroid:12.0.0-latest androidboot.redroid_width=1080 androidboot.redroid_height=1920 androidboot.redroid_dpi=480 androidboot.redroid_gpu_mode=host androidboot.redroid_gpu_node=/dev/dri/renderD128 androidboot.redroid_fps=120
```
