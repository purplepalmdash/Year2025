### 3. intel b60
Install b60 related  drivers:      

```
sudo usermod -aG render $USER
sudo add-apt-repository -y ppa:kobuk-team/intel-graphics-testing
sudo apt update
sudo apt install -y   intel-opencl-icd   libze-intel-gpu1 libze1   intel-level-zero-gpu   clinfo   intel-gsc   mesa-vulkan-drivers   intel-media-va-driver-non-free vainfo
# Intel完整的编译器套件
sudo apt install -y intel-basekit
sudo apt install vulkan-tools
sudo reboot
lspci | grep -i arc
lsmod | grep -E 'xe|i915'
dpkg -l | grep -E 'intel|ze|opencl|level-zero|clinfo' | cat
id
clinfo -l
vulkaninfo --summary | grep Intel
```
