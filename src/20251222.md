### 2. NitroGen Tips
Install git:     

```
https://git-scm.com/download/win
点击 "64-bit Git for Windows Setup" 开始下载。
```
Install python:    

```
推荐下载 Python 3.10 版本（兼容性好）：https://www.python.org/downloads/release/python-31011/
拉到页面最下方，找到 "Files"，选择 "Windows installer (64-bit)" 下载。
```
Set the pip's repository:     

```
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
mkdir Code
cd Code
git clone https://github.com/MineDojo/NitroGen.git
cd NitroGen

# 2. 创建虚拟环境 (这步很重要，防报错)
python -m venv venv
.\venv\Scripts\activate

# 3. 安装 PyTorch (显卡加速库)
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

pip install -e .

set HF_ENDPOINT=https://hf-mirror.com
huggingface-cli download nvidia/NitroGen ng.pt --local-dir . --local-dir-use-symlinks False

```
