### 3. kimi
Installation Steps:      

```
  sudo apt install -y python3-pip
  pip install --user kimi-cli --break-system-packages
```
Edit the profile via(`vim ~/.kimi/config.toml`):     

```
default_model = "nim-kimi"
default_thinking = false

[models.nim-glm]
provider = "nim"
model = "z-ai/glm4.7"
max_context_size = 131072

[models.nim-minimax]
```
