### 2. nixos(legacy grub)
Install steps:      

```
sudo fdisk -l
sudo parted /dev/vda -- mklabel msdos
sudo parted /dev/vda -- mkpart primary 1MB -2GB
sudo parted /dev/vda -- set  1 boot on
sudo parted /dev/vda -- mkpart primary linux-swap -2GB 100%
sudo mkfs.ext4 -L nixos /dev/vda1
sudo mkswap -L sway /dev/vda2
sudo mount /dev/disk/by-label/nixos /mnt
sudo swapon /dev/vda2
sudo nixos-generate-config --root /mnt
sudo vim /mnt/etc/nixos/configuration.nix 
sudo nixos-install 
nixos-enter --root /mnt -c 'passwd dash'
sudo nixos-enter --root /mnt -c 'passwd dash'
```
### 3. docker on rockylinux
Step:      

```
# 添加 Docker 官方仓库（CentOS 版对 Rocky 兼容）
sudo dnf config-manager --add-repo https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
# 将仓库地址替换为阿里云镜像（关键加速步骤）
sudo sed -i 's+download.docker.com+mirrors.aliyun.com/docker-ce+' /etc/yum.repos.d/docker-ce.repo
# 更新缓存
sudo dnf makecache
sudo dnf install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
sudo systemctl enable --now docker
sudo systemctl status docker
```
Be sure to disable firewalld and selinux, then you could run redroid.   
### 4. binding problem
`ssh -p 16522 -R 0.0.0.0:28082:127.0.0.1:8080 dash@xxx.xx.com ` only listens `127.0.0.1`, for listening `0.0.0.0`, do following modification.    

```
编辑 sshd 配置：
sudo vim /etc/ssh/sshd_config
找到或添加下面这行（推荐 clientspecified，更灵活）：
GatewayPorts clientspecified
# 或者直接允许所有：
# GatewayPorts yes
重启 sshd：
sudo systemctl restart sshd
```
