### 2 celadon build docker images
Steps:    

```
git clone https://github.com/projectceladon/celadon-build-environment
docker build celadon-build-environment
docker tag <ID> celadon-build-environment:latest
cd ~/SourceCodeOfCeladon
sudo docker run -it -v `pwd`:/src celadon-build-environment /bin/bash
```
### 3. build cic in docker
In docker instance, you should manually add locale, and use python2 instead of python3, because the docker instance is based on ubuntu22.04, then you could trigger build:    

```
dash@aiBox:/media/sda/celadon/cic/cic$ sudo docker run -it -v `pwd`:/src celadon-build-environment /bin/bash                                                                                  
celadon@d7ce6465194a:~$ cd /src
```
