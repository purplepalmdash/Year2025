### 2. ubuntu20.04 bisheng
Server版本安装，最小化，非图形界面，而后：     

```
sudo apt update -y
sudo apt upgrade -y
sudo dpkg --add-architecture i386
sudo add-apt-repository ppa:ubuntudde-dev/stable
sudo apt-get install ubuntudde-dde
sudo apt install -y gnome-terminal sakura cups
scp -r dash@192.168.1.208:~/hacker .
cd hacker/modified_bisheng
sudo apt install ./*.deb
sudo chown -R root:root /usr/lib/cups/filter
sudo chown -R root:root /usr/lib/cups/filter/gstobmp
sudo systemctl restart cups
```

