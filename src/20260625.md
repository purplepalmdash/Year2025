# 20260625
### 1. Intel dg1 on arm64
Issue:    

```
[   11.496106] xe 0000:09:00.0: Adding to iommu group 17
[   11.496159] xe 0000:09:00.0: Your graphics device 4909 is not officially supported
               by xe driver in this kernel version. To force Xe probe,
               use xe.force_probe='4909' and i915.force_probe='!4909'
               module parameters or CONFIG_DRM_XE_FORCE_PROBE='4909' and
               CONFIG_DRM_I915_FORCE_PROBE='!4909' configuration options
```
Added :     

```
$ sudo vim /etc/default/grub
GRUB_CMDLINE_LINUX_DEFAULT="console=tty0 net.ifnames=0 biosdevname=0 xe.force_probe='4909'  i915.force_probe='!4909'"
$ sudo update-grub2
```
