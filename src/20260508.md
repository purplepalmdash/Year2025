### 2. ollama related
Specify gpu 1 for running:      

```
sudo docker run -d --gpus '"device=1"'   -v /home/dash/ollama:/root/.ollama   -p 11434:11434   --name ollama   --restart always   ollama/ollama
```
Test:     

```
curl http://localhost:11434/api/generate -d '{
  "model": "gemma4:e2b",
  "prompt": "讲个关于程序员的笑话，300字。",
  "stream": false
}'
```
Remove unnecessary models:      

```
sudo docker exec -it ollama bash
ollama list
ollama rm gemma4:e4b
```
