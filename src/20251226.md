### 1. Building Kernel(tag-origin)
Build/install kernel with -origin:    

```
$ cp /boot/config-$(uname -r) .config: 复制当前配置。
$  make olddefconfig: 以默认值更新配置以适应新内核版本。
$  ./scripts/config --set-str LOCALVERSION -origin: 设置内核后缀为 -origin。
$ scripts/config --set-str SYSTEM_TRUSTED_KEYS ""
$ scripts/config --set-str SYSTEM_REVOCATION_KEYS ""
$ make -j$(nproc)
$ sudo make modules_install
$ sudo make INSTALL_MOD_STRIP=1 modules_install
$ sudo make install
```
After reboot:     

```
$ uname -a
Linux nuc11test 6.18.0-rc4-origin+ #1 SMP PREEMPT_DYNAMIC Fri Dec 26 01:22:30 UTC 2025 x86_64 x86_64 x86_64 GNU/Linux
```

```

```
