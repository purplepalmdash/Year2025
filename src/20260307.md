# 20260307
### 1. ACE-Step-1.5
Install uv:      

```
curl -LsSf https://astral.sh/uv/install.sh | sudo env UV_INSTALL_DIR=/usr/local/bin sh
```
Then:     

```
cd Code
git clone https://github.com/ACE-Step/ACE-Step-1.5.git
cd ACE-Step-1.5
uv sync

```
Clone llm:     

```
pip install modelscope

# 示例（根据实际模型 id 调整）
modelscope download ACE-Step/Ace-Step1.5 --local_dir ./models
modelscope download ACE-Step/acestep-5Hz-lm-4B --local_dir ./models/acestep-5Hz-lm-4B

mkdir checkpoints
mv models/* checkpoints

$ ls checkpoints/
acestep-5Hz-lm-1.7B  acestep-5Hz-lm-4B  acestep-v15-turbo  config.json  configuration.json  Qwen3-Embedding-0.6B  README.md  vae

```

Configure:     

```
$ vim acestep/llm_inference.py 

self.llm = LLM(
    ...,
    enforce_eager=True,          # vLLM/nanovllm 风格的写法
    ...,
)
```
Change privilege:    

```
# 1. 创建目录（如果不存在）
mkdir -p ~/.cache/huggingface/modules

# 2. 递归赋予当前用户（dash）完全权限
chmod -R u+rwx ~/.cache/huggingface

# 或者更激进（如果上面不行，给 everyone 写权限，单机使用无所谓）
chmod -R 777 ~/.cache/huggingface
```
Start via:      

```
uv run acestep --language zh --config_path acestep-v15-turbo --lm_model_path acestep-5Hz-lm-4B --init_service true --server-name 0.0.0.0
```
