### 1. waydroid workingtips
Install via:     

```
tar xvf waydroiddebs.tar
cat /etc/apt/sources.list 
......
deb [trusted=yes] file:///home/test/waydroiddebs/ ./
......
apt update -y
apt install -y waydroid

# extract the /var/lib/waydroid
sudo tar xzvf var_lib_waydroid.tar.gz -C / --same-owner --same-permissions
# extract the data 
sudo tar xzvf local_share_waydroid.tar.gz \
  -C /home/test/.local/share/ \
  --strip-components=4 \
  --same-permissions
waydroid session start

$ sudo waydroid shell
```

### 2. waydroid on vm
change to swiftshader:    

```
You can force Waydroid to run without GPU acceleration by modifying the waydroid configuration file:

nano /var/lib/waydroid/waydroid.cfg
Add the following lines in the [properties] section:

ro.hardware.gralloc=default
ro.hardware.egl=swiftshader
Apply the configuration with:

sudo waydroid upgrade -o
```
