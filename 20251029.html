3. fedora bridge

Change to eth0:

grubby --update-kernel="/boot/vmlinuz-6.17.1-300.fc43.aarch64" --args="net.ifnames=0 biosdevname=0"

Steps:

nmcli con down "Wired connection 1"
nmcli con add type bridge con-name bridge-br0 ifname br0
nmcli con add type ethernet con-name bridge-slave-eth0 ifname eth0 master bridge-br0 slave-type bridge
nmcli con mod bridge-br0 ipv4.method auto ipv6.method auto
nmcli con mod bridge-br0 bridge.stp no
nmcli con up bridge-br0
nmcli con up bridge-slave-eth0
nmcli device status

Be sure to disable power hibernation.
